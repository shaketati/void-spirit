# Void Spirit - Solar System Explorer

A visually stunning, immersive web experience featuring a stylized solar system.

## Features
- **Immersive Void Theme**: Interactive particle effects and atmospheric visuals.
- **Solar System Explorer**: Interactive 3D representation of the solar system using Three.js.
- **Detailed Planets**: Procedurally generated textures for gas giants, rocky planets, and the Sun.
- **Camera Tracking**: Smooth camera transitions and tracking that follows planets as they orbit.
- **Information Panel**: Detailed information about each planet in Japanese.

## Technologies
- **Three.js**: 3D rendering engine.
- **GSAP**: Animation library.
- **TailwindCSS**: Utility-first CSS framework.
- **Vanilla JavaScript**: No build step required.

## How to Run
Simply open `index.html` in your browser. No installation needed.

## Deployment
This project is ready for static hosting (Vercel, Netlify, GitHub Pages).
Just push to GitHub and import the repository.
